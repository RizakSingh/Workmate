# WorkMate AI

Full-stack AI assistant for daily work tasks.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Node.js + Express
- Docker

## Phases
- Phase 0: Project foundation + Docker
- Phase 1: Auth + MongoDB
- Phase 2: AI tools
- Phase 3: Dashboard + History
- Phase 4: Payments

## Run locally
docker compose up --build
